## Differences between interfaces and types in TypeScript 
Object structure এ interface  ব্যবহার করা হয়ে থাকে।
Type object ছাড়া Union, interaction, primitive alias এ ব্যবহার করে।

interface extended করা যায়।
কিন্তু Type এর ক্ষেত্রে আলাদাsyntax লাগে।

## Example of using union and intersection types in TypeScript

ইউনিয়ন মানে হলো একটাই ভ্যারিয়েবল, কিন্তু একাধিক টাইপের যেকোনো একটি হতে পারে।


Intersection মানে হলো দুটি টাইপকে একসাথে মিশিয়ে একটা নতুন টাইপ বানানো—যেখানে দুইটার সব প্রপার্টিই থাকতে হবে।
